# Programming Language Predictor

This project predicts the programming language of a code snippet using a Naive Bayes machine learning model.

## Technologies Used
- Python
- Scikit-learn
- Flask
- HTML
- CSS

## Features
- Predict programming language from code snippet
- Web interface using Flask
- Machine learning model using Naive Bayes

## Supported Languages
- Python
- C
- Java
- JavaScript

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the app:

python app.py

Open browser:

http://127.0.0.1:5000
